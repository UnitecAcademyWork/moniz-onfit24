1. ⛔️ Valida dados obrigatórios **dataNascimento**, **peso**, **altura**, **genero** e **objectivos**
1. ⛔️ Valida que o campo **dataNascimento**, **peso**, **altura**, **genero** é válida
1. ⛔️ Gera dados do utilizador com os dados fornecidos
1. ⛔️ Retorna 201
